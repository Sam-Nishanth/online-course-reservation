🚀 Online Course Reservation System
📌 Project Overview

A full-stack web application for managing online course reservations with role-based access control.

👥 Roles

Admin – Manage users and courses

Teacher – Create and manage courses

Student – View courses, enroll, and track progress

🛠 Tech Stack

Frontend

React.js

Axios

React Router

Backend

Flask

SQLAlchemy

SQLite

🎯 Features

User Authentication (Login/Register)

Role-based Dashboard

Course Listing

Course Details with 12-Week Schedule

Enroll System

Progress Tracking

Admin: Manage Users & Courses
